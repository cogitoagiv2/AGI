## ⚠️ Legacy Cogito (Deprecated)

The original **Cogito** framework (Go-based agentic system) introduced powerful primitives for reasoning, tool execution, and structured LLM workflows .

However, over time, several issues became apparent:

### ❌ Problems in the Old Version

* Inconsistent evaluation and execution behavior
* Implicit assumptions in agent pipelines
* Edge-case failures in tool orchestration
* Lack of a strict, enforceable contract
* Difficult reproducibility across runs and models

These limitations made it unsuitable as a **reliable foundation for AGI evaluation and research-grade experimentation**.

## 🆕 Why AGI v2 Exists

To address these issues, the framework has been **rebuilt from the ground up** in a new repository with a completely redesigned core:

### ✅ What Changed

* **New Contract (AGIv2)**
  A strict, versioned specification governing all evaluations

* **Deterministic Execution Model**
  Reduced ambiguity and improved reproducibility

* **Explicit Schemas & Validation**
  No hidden behavior or undefined states

* **Separation of Concerns**
  Clean boundaries between:

  * Evaluation
  * Agents
  * Metrics
